# 🏏 IPL Fan Arena

**Free fantasy cricket. Real-time. Pure skill.**

IPL Fan Arena is a Dream11-style fantasy cricket platform where users compete with friends in real-time — without any money involved. It focuses on skill-based gameplay, fast updates, and an engaging competitive experience for IPL fans.

---

## 🚀 Features

### ⚡ Real-Time Gameplay
- Live leaderboard updates using WebSockets  
- Sub-200ms response time for smooth experience  
- Instant UI updates on database changes  

### 🧠 Smart Scoring System
- Dynamic points-based scoring  
- Automated point calculation using cron jobs  
- No manual intervention required  

### 🏆 Lobby System
- **Single Match Lobby**  
  - Compete based on a single match  
  - Quick and fast-paced gameplay  

- **Tournament Lobby**  
  - Join with friends  
  - Compete across the entire IPL season  
  - Cumulative leaderboard for long-term competition  

### 🔐 Security & Performance
- Rate limiting & bot protection using Redis  
- Row-Level Security (RLS) for secure data access  
- Optimized queries with B-tree indexing in PostgreSQL  

### 🎨 UI/UX
- Clean and responsive design  
- Constraint-driven UI to prevent invalid actions  
- Smooth navigation and gameplay flow  

### 🐳 Developer Experience
- Containerized using Docker  
- Monorepo architecture for better code management  
- Automated workflows with GitHub  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS  
- **Backend:** Node.js  
- **Database:** Supabase (PostgreSQL)  
- **Real-time:** WebSockets  
- **Caching & Security:** Redis  
- **Automation:** Cron Jobs  
- **Auth:** Google Authentication  
- **Deployment & DevOps:** Docker, Vercel, GitHub
