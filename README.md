# IPL Fan Arena 🏏
An automated Fantasy Cricket platform for private group competition.

## 🏗️ System Architecture
- **Frontend:** Next.js 15 (App Router)
- **Backend:** Server Actions & Node.js
- **Database:** Supabase (PostgreSQL)
- **DevOps:** Docker & GitHub Actions

## 🚀 Local Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
